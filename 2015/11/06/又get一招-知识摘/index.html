又get一招--知识摘

学习工作中总能碰到一些不错的解决方案，用过一次，后来忘记了，开此笔记做一点记录，希望以后碰到一些问题，能在这个列表中翻到吧。

新技能 新元素

1 HTML5 input的新技能属性
首先 了解的是number 属性，还记得以前做一个input输入框要求只能输入数字，然后写js处理方法来做到，那个时候不知道有这些新特性真是…waste life.

简单列一下 HTML5 input types的新元素

  • color
  • date/datetime/datetime-local/month/week/time
  • number
  • range
  • search
  • tel/email/url

以后在浏览器不受限的前提下，多用用这些新东西，挺好的。
